About

This is a simple yet powerful Library Management System built using HTML, CSS, and JavaScript.

Key Features

  • Book Management: Add, edit, and delete book records including title, author, ISBN, publication date, category, and availability status.
  • Borrow & Return System: Users can borrow and return books with automatic status updates and borrow history tracking.
  • User Authentication: Basic login functionality to personalize user experience and manage borrowed books.
  • Search & Filter: Advanced live search to filter books by title, author, ISBN, category, or status for easy navigation.
  • Persistent Data Storage: Utilizes localStorage for saving data locally, ensuring information persists across browser sessions without server-side requirements.
  • Responsive and Accessible UI: Fully responsive design compatible with desktops, tablets, and mobiles, adhering to accessibility standards.
  • Dark Mode: Toggle light/dark themes for user comfort during day and night usage.
  • Dynamic Statistics: Real-time counters for total books, available books, borrowed books, and active users.
  • Category Management: Automatically generate and manage book categories based on user inputs for flexible organization.
  • CSV Export & Import: Export library data to CSV for backup or external analysis, and import data to quickly restore or add records.

Technologies Used

  • HTML5: Semantic markup for structured content.
  • CSS3 & Bootstrap: Modern styles and responsive design framework.
  • JavaScript: DOM manipulation, event handling, data storage, and dynamic rendering.
  • localStorage API: Client-side persistent storage without backend dependencies.

Future Enhancements

  • Integration with backend databases for centralized data management.
  • User roles and permissions to secure sensitive operations.
  • Email and SMS notifications for book due dates and system updates.
  • Enhanced reporting tools with PDF export functionality.
  • Integration with barcode scanners for easy book check-in/out.
  • Mobile app version with offline sync capabilities.

This project not only provides a practical tool for managing book collections but also serves as a valuable educational resource, illustrating key concepts in front-end development, user experience design, and client-side data management.
